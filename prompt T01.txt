Buatlah program untuk menghitung biaya pengiriman paket Del-Express berdasarkan kota tujuan dan berat paket.
Buatlah program untuk sistem Del-Express berdasarkan cerita soal. Ucok dan Butet ingin mengirim paket ke beberapa kota tujuan. Program menerima input secara berulang berupa kode kota tujuan dan berat paket Butet. Jika kode kota yang dimasukkan adalah END, maka program berhenti. Untuk setiap data pengiriman, program harus menentukan nama kota berdasarkan kode kota, menentukan ongkos kirim per kilogram sesuai kota tujuan, menghitung berat paket Ucok berdasarkan perbandingan ukuran dus pada soal, menghitung total berat paket Butet dan Ucok, menghitung total ongkos kirim, menentukan promo yang diperoleh berdasarkan aturan promo pada soal, lalu menampilkan struk pembayaran. Struk pembayaran harus berisi kota tujuan, berat paket Butet, berat paket Ucok, total berat, total ongkos kirim, dan informasi promo yang diperoleh. Gunakan hanya data kode kota, daftar ongkos kirim, aturan promo, dan perbandingan berat paket yang terdapat pada gambar soal, serta jangan membuat asumsi tambahan di luar informasi yang diberikan.
